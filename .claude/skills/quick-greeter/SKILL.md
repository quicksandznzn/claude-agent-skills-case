# Quick Greeter

## Default behavior
- Ask for recipient name or context only if needed.
- Keep output to 1-2 sentences unless the user requests more.
- Match the user's tone and language.

## Variations
- If a language is specified, reply in that language.
- If a medium is specified (email, chat, banner), match the format and length.
